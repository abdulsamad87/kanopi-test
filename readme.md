# Kanopi WordPress & Gutenberg Code Samples

This repository contains production-ready WordPress and Gutenberg examples created to demonstrate real-world development practices.

## Included Examples

### 1. Full Site Editing (FSE) Template
- Block-based '.html' template
- Dynamic current date rendered via shortcode
- Query loop for posts

### 2. Advanced SCSS Styling
- Responsive grid layout
- CSS animations
- Accessibility-focused features
- Reduced motion support

### 3. Custom Gutenberg Block (No ACF)
- Built with React
- Uses block.json
- Inspector controls and editable attributes

### 4. WordPress PHP Customizations
- Custom shortcode
- WordPress behavior modification via filters
- Security-focused enhancements

## Notes
These examples reflect how I structure and write code for real production projects, focusing on clarity, accessibility, maintainability, and performance.
